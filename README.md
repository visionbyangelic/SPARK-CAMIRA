## 🧠 About SPARK
The **SPARK Academy** is a MICCAI-endorsed program designed to train African AI experts in medical imaging. It uses a collective intelligence approach—combining radiology, neurology, and computer science—to turn AI knowledge into practical action, specifically focusing on **glioma brain tumor segmentation**.
